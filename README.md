# MNIST Digit Classifier (From Scratch)

This repository implements a handwritten digit classifier using the MNIST dataset.  
The neural network is implemented from scratch in Python (NumPy), including forward propagation, backpropagation, and evaluation.

## Files
- `src/train.py` — main training/evaluation script
- `src/experiments.py` — alternate implementation / experiments

## Requirements
- Python 3.x
- numpy, pandas, matplotlib

Install:
```bash
pip install -r requirements.txt
